# 정보보안 3요소 CIA

- Confidentiality : 허가된 사용자에 대해서만 확인이 가능해야 함
- Integrity : 허가된 사용자에 대해서만 수정 및 삭제가 가능하고 그 외의 사용자에게는 변경되어선 안 됨
- Availability : 허가된 사용자가 언제나 접근 가능해야 함
