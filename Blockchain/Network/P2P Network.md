# Introduction

중앙 서버 없이 각 단말이 서로 동등한 입장에서 통신을 하는 네트워크. 각 단말은 서버이기도 하고 동시에 클라이언트가 될 수 있음.<br/>
비트코인 네트워크는 전체 노드가 블록과 트랜잭션 교환을 위한 P2P 네트워크를 공동으로 관리할 수 있음.<br/>
합의 알고리즘은 네트워킹를 커버하지 않기 때문에 비트코인 프로그램은 채굴자나 별도의 네트워크 프로토콜을 사용 함.
